What is FERDA?

FERDA (Fast Extremal Region Detector of Ants) is a method for tracking multiple objects hardly distinguishable by the naked eye (ants, zebrafish, bugs, mice...). Its main purpose is to ease laboratory biological experiments - a scientist now has to provide only a video sequence and number of annotations to receive full tracking information. An elaborated graphical user interface with several tools is available to check output and correct potential errors. We believe that this system will help many research groups to achieve better results in their field.

Code Availibility

You may be wondering why there is no source code available in this repository. The reason behind this is that we are approaching very final stage of development and do not want to rush things. Due to the nature of our work, we want to make sure that released code is free of bugs and entiraly reliable.
